# 🕌 منصة الإمام مالك التعليمية

منصة تعليمية متكاملة لطلاب كلية الإمام مالك، تعمل داخل تطبيق تيليجرام باستخدام Mini App، لتكون المرجع الأول الموحد للوصول إلى جميع المحتويات الدراسية والقنوات الرسمية والمجموعات الطلابية، مع كناشة خاصة للملاحظات والفوائد.

## 🎯 الهدف
تجميع وتنظيم المصادر التعليمية في مكان واحد داخل تيليجرام، وتوفير أداة شخصية (كناشة المتفقه) لتسجيل الفوائد، بحيث يحصل الطالب على تجربة سلسة ومنظمة دون مغادرة التطبيق الذي يستخدمه يومياً.

---

## ⚙️ الميزات الرئيسية

| الميزة | الوصف |
|--------|-------|
| 📚 **دليل القنوات** | قنوات الكلية الرسمية، قنوات دعوية وقرآنية، ومجموعات طلابية، مصنفة ومرتبة، بضغطة واحدة تنتقل للقناة |
| 📝 **كناشة المتفقه** | مساحة خاصة لكل طالب لتسجيل الفوائد والملاحظات مع تصنيفها (فقه، لغة، أصول فقه، حديث، سيرة، فائدة عامة) وإمكانية التعديل والحذف |
| 🔒 **خصوصية تامة** | الكناشة خاصة بكل مستخدم، لا يراها أحد غيره، ومؤمنة بالتحقق من هوية تيليجرام |
| 🛠️ **لوحة تحكم** | للمشرف فقط: إضافة وتعديل وحذف القنوات والمجموعات بسهولة من داخل Mini App |
| 🚀 **سريعة وخفيفة** | تصميم نظيف بدون تعقيدات، يعمل بسلاسة داخل تيليجرام |
| 🔐 **أمان عالي** | جميع الطلبات موثقة عبر `initData` من تيليجرام، وفصل صارم للصلاحيات |

---

## 🧱 التقنيات المستخدمة

| المجال | التقنية |
|--------|---------|
| **الخادم** | Python 3.11+ · FastAPI |
| **بوت تيليجرام** | aiogram 3.x |
| **قاعدة البيانات** | SQLite (تطوير) / PostgreSQL (إنتاج) · SQLAlchemy |
| **الواجهة الأمامية** | HTML · CSS (Tailwind) · Vanilla JavaScript |
| **الأمان** | تحقق من `initData` · صلاحيات مشرف عبر `.env` |
